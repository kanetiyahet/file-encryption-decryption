# File Encryption Pro 🔐

## 📌 Overview
A secure file encryption and decryption tool using AES-256 algorithm with a modern GUI interface.

## 🚀 Features
- AES-256 encryption
- File & folder encryption
- Secure key-based protection
- File shredding (unrecoverable deletion)
- Steganography support
